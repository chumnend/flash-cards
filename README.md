### Prerequistes
- Install Python 3
- Install Pip 

### Start
Setup virtual environment,
```
python3 -m virtualenv venv
source venv/bin/activate
```

Install dependencies,
```
pip install -r requirements.txt
```

Run any migartions,
```
python manage.py migrate
```

Start the app,
```
python manage.py runserver
```
